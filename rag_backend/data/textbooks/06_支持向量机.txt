# 第六章 支持向量机

## 6.1 间隔与支持向量

支持向量机（SVM, Support Vector Machine）是一种经典的二分类模型，其核心思想是寻找一个能将两类样本正确分开且间隔最大的超平面。

超平面（Hyperplane）：在 n 维空间中，超平面是维度为 n-1 的子空间，方程为 w^T·x + b = 0。

间隔（Margin）：分类超平面到最近训练样本的距离。SVM 的目标是最大化这个间隔，因为间隔越大，分类的置信度越高，泛化能力越好。

支持向量（Support Vectors）：距离分类超平面最近的训练样本点。这些点决定了决策边界的位置——去掉其他样本点不会改变超平面，但去掉支持向量会改变超平面。

SVM 的优化目标：
minimize (1/2)||w||²
subject to y_i(w^T·x_i + b) ≥ 1, for all i

## 6.2 对偶问题与 SMO 算法

对偶问题（Dual Problem）：通过拉格朗日对偶性，将 SVM 的原始优化问题转化为对偶形式。对偶问题有两个好处：一是更容易求解；二是自然地引入了核函数。

对偶问题最终需要求解一组拉格朗日乘子 α_i，每个 α_i 对应一个训练样本。α_i > 0 对应的样本就是支持向量。

SMO 算法（Sequential Minimal Optimization）：将大规模的二次规划问题分解为一系列最小规模的子问题（每次只优化两个 α），通过迭代求解实现高效训练。SMO 是实践中最常用的 SVM 求解算法。

## 6.3 核函数

当数据线性不可分时，SVM 通过核函数（Kernel Function）将原始数据隐式地映射到高维特征空间，在高维空间中寻找线性可分的超平面。

核技巧（Kernel Trick）：通过核函数直接计算高维空间中两个向量的内积，而无需显式地做特征映射。这避免了高维计算带来的"维数灾难"。

常用核函数：
- 线性核：K(x, x') = x^T·x'。不做任何映射，适用于线性可分数据
- 多项式核：K(x, x') = (γ·x^T·x' + r)^d。d 为多项式的度数
- RBF 核（高斯核）：K(x, x') = exp(-γ||x - x'||²)。最常用的核函数，能将数据映射到无穷维空间，适用于大多数非线性分类问题
- Sigmoid 核：K(x, x') = tanh(γ·x^T·x' + r)

选择核函数的一般原则：先尝试线性核，如果效果不好再尝试 RBF 核。RBF 核的参数 γ 通常通过交叉验证来选择。

## 6.4 软间隔与正则化

软间隔 SVM（Soft Margin SVM）：允许部分样本违反间隔约束（即允许少量样本被错分），从而提高模型的实用性。

优化目标变为：
minimize (1/2)||w||² + C·Σξ_i
其中 ξ_i 是松弛变量，表示第 i 个样本的违反程度；C 是惩罚参数，控制对错分的容忍度。

C 的选择：C 越大，对错分的惩罚越重，模型越复杂（容易过拟合）；C 越小，允许更多错分，模型越简单（可能欠拟合）。通常通过交叉验证来选择 C。

## 6.5 支持向量回归

支持向量回归（SVR, Support Vector Regression）将 SVM 的思想扩展到回归任务。SVR 容忍预测值与真实值之间一定范围内的偏差（ε-不敏感损失），只有超出这个范围的偏差才被计入损失。
